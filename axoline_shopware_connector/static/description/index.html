Axoline Shopware Connector

Connect Shopware 6 to Odoo 19: synchronize products, categories, customers, and orders inside your Odoo database—no separate middleware server required.

Key features

Products & categories

Import and align products and product categories between Shopware and Odoo. Export paths support pushing Odoo data back to Shopware where configured.

Customers

Synchronize customers and link them to Odoo contacts for a single customer view across ecommerce and ERP.

Orders & sales

Import sales orders from Shopware into Odoo, including linkage to the Shopware order record. Order and payment metadata can be reflected on the Odoo sales order for traceability.

Automation

Use scheduled jobs (cron) for recurring sync and run manual operations from the Shopware backend screens in Odoo.

How it works

The module adds a Shopware application menu and backend records where you store your Shopware 6 API credentials (OAuth client). After a successful connection test, you can import and export the supported objects according to your settings.

Communication uses the Shopware 6 Admin API. Your Shopware shop must be reachable from the Odoo server (network / firewall). Operation requires valid API credentials created in Shopware.

Requirements

  • Odoo 19.0 (Community or Enterprise), with standard apps as declared in the module dependencies.
  • Shopware 6 with Admin API access and OAuth credentials (client ID / secret).
  • Outbound HTTPS from Odoo to your Shopware base URL.

Data & external service

This connector exchanges business data (e.g. products, customers, orders) between your Odoo instance and your Shopware shop via the API you configure. You are responsible for lawful processing, Shopware terms, and any data protection obligations that apply to your deployment.
